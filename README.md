## Boilerplate для frontend-разработки

За основу взята сборка: https://github.com/rokoroku/react-redux-typescript-boilerplate

Дополнительно добавлены:
* Apollo
* GraphQL
* Office UI Fabric

Запуск проекта:
    $ git clone https://github.com/pyshopml2/iw-frontend
    cd iw-frontend
    npm install
    npm start
